# Python website
The website is built using the framework called [Flask](https://flask.palletsprojects.com/en/3.0.x/).

## How to use
1. Install the modules.
2. Run main.py
3. Visit: http://127.0.0.1:5000

# Modules installed

1. 
```
pip install flask
```
2. 
```
pip install flask-login
```
3. 
```
pip install flask-sqlalchemy
```
